## 📋 О проекте

**Тест Памяти** - это интерактивная игра-тест для диагностики кратковременной памяти и внимания. Программа оценивает когнитивные способности пользователя и выдаёт результат по трём категориям: **Хорошо**, **Нормально** или **Плохо**.

Игра основана на классическом принципе "Найди пару" (мемори), но с медицинским уклоном - результаты сохраняются и анализируются для выявления возможных когнитивных нарушений.

### 🎯 Цель тестирования

- Оценка кратковременной зрительной памяти
- Диагностика внимания и концентрации
- Выявление возможных когнитивных нарушений
- Мониторинг динамики когнитивных способностей

## ✨ Возможности

### 🎮 Три типа тестов

1. **Числа** - классический вариант с цифрами
2. **Цвета** - нужно запомнить расположение цветных квадратов
3. **Формы** - комбинация цвета и геометрической фигуры (круг, квадрат, треугольник, ромб)

### 📊 Четыре уровня сложности

| Размер сетки | Количество пар | Сложность |
|-------------|---------------|-----------|
| 2×2         | 2 пары        | Очень легко |
| 4×4         | 8 пар         | Легко      |
| 6×6         | 18 пар        | Средне     |
| 8×4         | 16 пар        | Сложно     |

### 📈 Система оценки

Результат оценивается по трём критериям:
- **Количество ходов** (чем меньше, тем лучше)
- **Количество ошибок** (чем меньше, тем лучше)
- **Время прохождения** (чем быстрее, тем лучше)

**Категории оценки:**
- 🟢 **Хорошо** - отличная память, результат выше нормы
- 🟡 **Нормально** - результат в пределах возрастной нормы
- 🔴 **Плохо** - результат ниже нормы, рекомендуется консультация специалиста

### 💾 Статистика

- Все результаты сохраняются в Excel-файл (`game_stats.xlsx`)
- Отслеживание динамики по датам
- Анализ по типам тестов и уровням сложности
- Просмотр статистики прямо в игре

🎮 Как играть
Правила игры
Перед вами поле с закрытыми карточками

Кликайте на карточки, чтобы открыть их

Нужно найти две одинаковые карточки (пару)

Когда пара найдена, карточки остаются открытыми

Цель - найти все пары за минимальное количество ходов

Интерфейс
Ходы - количество сделанных попыток (левый верхний угол)

Ошибки - количество несовпавших пар (левый верхний угол, красным цветом)

Пары - сколько пар уже найдено (правый верхний угол)

Режим - текущий тип теста и сложность (центр вверху)

Управление
Мышь - клик по карточкам

ESC - выход в главное меню (во время игры)

ESC - выход из программы (в главном меню)

🛠️ Устранение неполадок
Ошибка "pygame.error: font not initialized"
Решение: Убедитесь, что у вас установлена последняя версия Pygame:

bash
pip install --upgrade pygame
Ошибка "ModuleNotFoundError: No module named 'pandas'"
Решение: Установите недостающую библиотеку:

bash
pip install pandas openpyxl
Не открывается файл статистики в Excel
Решение:

Закройте файл Excel перед новой игрой

Установите openpyxl: pip install openpyxl

Удалите повреждённый файл game_stats.xlsx (игра создаст новый)

Нет звука
Проверьте:

Включены ли звуки в настройках игры

Есть ли файлы в папке assets/sounds/

Не отключён ли звук в системе

Корректность формата файлов (WAV для звуков, MP3/OGG для музыки)

Игра тормозит
Рекомендации:
